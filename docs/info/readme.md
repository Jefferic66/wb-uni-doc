## 来由

集成 uView-UI1.x 和 ColorUI 的 uniapp 项目脚手架，再加上部分自己的组件

## 快速上手

### 1. 克隆项目

```sh
git clone https://gitee.com/jefferic/wb-uni-template.git
```

### 使用你喜欢的包管理器进行初始化

```sh
yarn init # npm init
```

### 2. 手动创建项目

```sh
yarn add wb-uni cl-uni html2canvas uview-ui   # npm install --save wb-uni cl-uni html2canvas uview-ui
```
