## uView 的内置样式

> **`uView-UI`** 中有很多很实用的内置样式，[文档](https://uviewui.com/components/common.html)

- 扩展的样式以 **`u-`** 开头
- 扩展的样式变量以 **`$u-`** 开头

## 本项目的内置样式

- 内置的样式以 **`wb-`** 开头
- 内置的样式变量以 **`$wb-`** 开头
- 扩展 uView 的样式：
  - .u-flex-col
  - .u-full
